Perspective wall View

The perspective wall is used to layout 2D information into 3D segmented wall. The front segment of the wall is used for viewing details. Whereas, the side segments will provide the context.

Based on the perspective wall concept we have built this variant. When the user moves courser to the focused cell, some extent of rows and column will be enlarged. The Table Lens: remove static control and fill display the neighbor context in the table.
